# TransformHub — Mobile App Mockup

Мобильный прототип приложения для медитации, программ развития и аудиоконтента в виде макета телефона.

## Структура проекта

```
AppTech/
├── index.html      # Главная страница
├── css/
│   └── styles.css  # Стили
├── js/
│   └── main.js     # Навигация и логика плеера
└── README.md
```

## Запуск

Откройте `index.html` в браузере (двойной клик или через Live Server / любой локальный сервер).

## Экраны

- **Today** — приветствие, цитата дня, «Продолжить обучение», ежедневная медитация, популярные программы
- **Programs** — поиск, категории (All, Mind, Body, Soul, Career), список программ
- **Program Detail** — описание программы Mind Mastery, кнопка «Start Quest», путь по дням, бейдж Quest Master
- **Audiowaves** — таймер сна (15/30/60 мин / ∞), категории, аудио: сон, медитации, дыхание
- **Community** — «племена» (Mind, Body, Love, Career), лента постов
- **Profile** — профиль пользователя, статистика (streak, время, завершённые), подписка, достижения, «Моя библиотека»

## Функциональность

- Переключение экранов через нижнюю навигацию и клики по карточкам
- Плавающий аудиоплеер при нажатии на аудио (плей/пауза, закрытие)
- Переключение активной категории (пилюли) и кнопок таймера сна
- Шрифт Inter, Tailwind CSS (CDN), Font Awesome иконки

## Технологии

- HTML5
- Tailwind CSS (CDN)
- Font Awesome 6
- Google Fonts (Inter)
- Vanilla JavaScript
